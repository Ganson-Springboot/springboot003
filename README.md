5.2管理员功能模块

学生管理，在学生管理页面中可以通过查看学号、密码、学生姓名、性别、出生日期、联系电话、班级等内容进行修改、删除等操作，如图5-6所示。还可以根据需要对图书分类管理进行修改或删除等详细操作，如图5-7所示。
